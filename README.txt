抖音用户信息采集器

这是一个Chrome浏览器扩展，用于采集抖音用户主页信息。

功能特点

- 自动采集用户信息（用户名、抖音号、联系方式等）
- 智能防检测的自动翻页（3-10秒随机间隔）
- 实时倒计时提示
- 数据持久化存储：
  - 基于抖音号去重
  - 自动追加新数据
  - 手动清空功能
- 支持筛选功能：
  - "仅显示手机号用户"按钮
  - "显示全部用户"按钮
- 数据导出功能：
  - 导出全部用户数据
  - 仅导出有手机号用户数据

数据采集结构

采集目标页面：抖音搜索结果页面
示例：https://www.douyin.com/search/[关键词]?type=user

数据持久化和导出说明

1. 数据持久化机制
   - 使用 Chrome Storage API 存储数据
   - 以抖音号为唯一标识符（主键）
   - 每次采集时：
     * 检查抖音号是否存在
     * 如果是新抖音号，直接追加
     * 如果抖音号已存在，更新该条数据
   - 数据永久保存，浏览器关闭后不会丢失
   - 仅在用户主动点击"清空数据"时重置
